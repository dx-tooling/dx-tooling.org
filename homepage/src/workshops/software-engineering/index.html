Curriculum Outline

Module 1 — Foundations

  • Architecture principles for evolving systems
  • Bounded contexts and module boundaries
  • Error handling and observability basics

Module 2 — Testing Strategy

  • Pyramids vs. trophies: choosing the right mix
  • Fast unit tests, resilient integration tests
  • CI hygiene and flake mitigation

Module 3 — Developer Experience

  • Local DX: scripts, linters, pre-commit, containers
  • Productive repos: structure, docs, templates
  • Change velocity without regressions

Module 4 — AI in the Loop

  • Effective prompts and guardrails
  • Agentic workflows for tests and refactors
  • Safety, reviews, and accountability
