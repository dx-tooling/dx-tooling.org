Starter Kit Highlights

Guided Setup Wizard

Multi-Language SEO URLs

Sensible PHP Setup & Config

Battle-tested conventions, configuration, and environment management tuned for long-term maintainability.

Ready-to-use Dev Helper Scripts

Multi-Language Support

Serve audiences worldwide with localisation baked into UI components, routing, and content governance.

Magic Links Authentication

Deliver frictionless sign-in flows with secure, token-based magic links supported out of the box.

Code Style Enforcement

Static Analysis Done Right

Catch entire classes of bugs early with PHPStan configured at max level and CI-ready reporting.
